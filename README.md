# SpeakSprint

一个零依赖的英语口语练习网站，直接在浏览器中完成：

- 自定义主题并用 AI 进行英语对话
- AI 先开场，并在每轮回复里引导你继续说
- 每次录音最长 90 秒，可随时停止并发送
- 本地保存你的录音文件，支持重放、AI 纠错、AI 润色
- 对你的回复进行评分：发音、语法、地道度、总分
- 对话结束后生成一次整轮评价

## 运行方式

这是一个纯前端项目，不需要安装依赖。为了让浏览器更稳定地访问麦克风和本地文件，建议用本地静态服务器启动。

例如：

```bash
cd /Users/yusanjin/EnglishLearning
python3 -m http.server 4173
```

然后打开：

[http://localhost:4173](http://localhost:4173)

## 使用说明

1. 输入你的 API Key。
2. 根据你使用的服务填写 `API Base URL`。
   - OpenAI 官方：`https://api.openai.com`
   - 兼容网关：填写对应地址
3. 选择主题预设，或切到“自定义主题”后填写练习场景。
4. 选择对话模型、转写模型，以及语音模型和人声。
5. 点击“开始练习”，AI 会先用英语开场。
6. 点击“开始录音”，最长可录 90 秒；你可以提前停止并发送。
7. 发送后会自动：
   - 通过你配置的 API 接口转写你的语音
   - 保存录音到浏览器本地 IndexedDB
   - 生成评分、纠错、润色
   - 让 AI 继续对话
   - 优先使用 OpenAI/兼容接口的 TTS 语音播放回复
8. 点击“结束并总结”可拿到本轮对话总评。

## 技术说明

- `index.html`: 页面结构
- `styles.css`: 页面视觉与响应式布局
- `app.js`: 录音、API 请求、评分、本地存储逻辑

## 注意事项

- 默认 API 基础地址已切换为 `https://api.gptsapi.net`，复制密钥后即可使用。
- 默认转写模型已改为兼容性更高的 `whisper-1`；如果你的网关支持别的模型，也可以手动修改。
- 前端录音会优先编码为标准 `WAV` 再上传，以提高对 OpenAI 兼容网关的音频解析兼容性。
- 语音回复优先调用 `/v1/audio/speech`；若失败，会回退到浏览器本地朗读。
- API Key 只保存在当前浏览器的 `localStorage` 中。
- 录音文件保存在当前浏览器的 `IndexedDB` 中，不会自动上传到你自己的服务器。
- “发音评分”目前基于语音转写后的文本语义和口语表现做近似评估，不等同于专业语音测评引擎。
